About the Game

This game is at a prototype level right now and does not have all the features that it would ideally have if fully coded correctly. This game is meant to be an educational game that would ideally be placed in a museum setting so that it will occupy the artistic space. Although the theme and story of the game is not necessarily directly related to art, its generative art within the game shows that the interaction can be an art form in itself. It does not have all the code yet so it is a bit bare.

Oh NO!

An explosion has occurred on a drilling rig, causing the rapid spilling of oil into the surrounding bodies of water. The oil is causing intense damage to the ocean ecosystem in the surrounding area and is rapidly spreading. Luckily, a group of scientists have created a team of Super Bacteria that like to break down oil in water. Help your own Super Bacteria clean up the ocean.
